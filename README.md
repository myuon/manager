## Setup Instructions

### 1. UV Setup
1. Download the UV software from the official website.
2. Follow the installation instructions provided.
3. Verify the installation by running `uv --version`.

### 2. Infisical Setup
1. Visit the Infisical GitHub repository.
2. Clone the repository to your local machine.
3. Follow the build instructions in the README file of Infisical.

### 3. Project Execution
1. Run `make build` to compile the project.
2. Start the server using `make start`.
3. Access the application at `http://localhost:8080`.
